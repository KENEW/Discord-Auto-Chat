# Discord-Auto-Chat
디스코드 서버에서 채팅을 자동으로 입력해줍니다.

## 주요 기능
- 미리 입력된 문자열을 채널 내에서 보내준다. (순서, 랜덤)
- (제작 예정) 제한된 문자 중 자연스럽게 문자를 만들어준다.
- (제작 예정) 채팅 딜레이 내에서 마지막으로 검색된 단어가 나오면 답장 기능과 함께 문자열을 보내준다

## 환경 설정
- 프로젝트 파일내에서 .env 생성 후

```
Discord_Auth_Token = ''
Chanel_End_Point = ''
```
